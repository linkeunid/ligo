# Ligo 1.0 Release Roadmap

## Current Status: 0.9 ✅

Ligo has reached 0.9 maturity with all core features implemented:
- Complete DI system (Singleton, Transient, Request-Scoped)
- Full HTTP routing with Guards, Pipes, Interceptors, Exception Filters
- Module-level lifecycle hooks
- Dynamic modules
- Built-in utilities (guards, pipes, interceptors, response helpers)

## Out of Scope (By Design)

- ❌ Runtime lazy loading - Not idiomatic in Go
- ❌ Decorators - Go doesn't have decorators; uses builder pattern
